## Description

NestJS_REST_API" is a robust backend framework leveraging TypeScript for building efficient and scalable RESTful APIs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
